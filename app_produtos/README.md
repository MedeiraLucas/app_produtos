## Funcionalidades

- Listagem de produtos
- Cadastro de novo produto
- Tela de detalhes
- Edição de produto
- Exclusão de produto
- Navegação entre telas com Navigator
- Retorno de dados com Navigator.pop()

## Estrutura

```bash
lib/
├── main.dart
├── models/
│   └── produto.dart
└── screens/
    ├── lista_produtos.dart
    ├── cadastro_produto.dart
    └── detalhe_produto.dart
```

## Como executar

```bash
flutter pub get
flutter run
```
