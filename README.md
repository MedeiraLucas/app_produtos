📱 App de Produtos

Aplicativo desenvolvido em Flutter para gerenciamento de produtos com operações completas de CRUD.

🚀 Sobre o Projeto

Este projeto foi desenvolvido como parte da formação em Análise e Desenvolvimento de Sistemas, com o objetivo de aplicar conceitos fundamentais do desenvolvimento mobile utilizando Flutter.

O sistema permite o gerenciamento de produtos de forma simples e eficiente, simulando funcionalidades comuns em aplicações reais.

✨ Funcionalidades
📋 Listagem de produtos
➕ Cadastro de novos produtos
🔍 Visualização detalhada
✏️ Edição de produtos
🗑️ Exclusão de produtos
🔔 Feedback visual com SnackBar
🧠 Tecnologias Utilizadas
Flutter
Dart
Material Design
🧩 Conceitos Aplicados
Navegação entre telas (Navigator.push / Navigator.pop)
Gerenciamento de estado com StatefulWidget
Organização em camadas (models e screens)
Manipulação de listas em memória
Validação de formulários
